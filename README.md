# MCM
MCM is a free Minecraft Mod that adds new Ores to the game.
----------
**Current Items:**
- Ruby
- Black Opal
- Raw Black Opal
- Citrine
- Raw Citrine

**Current Blocks:**
- Ruby Block
- Black Opal Block
- Black Opal Ore
- Citrine Block
- Citrine Ore

**Planned Features:**
- [x] Deepslate Ores (WIP)
- [ ] Ore Generation
- [ ] More Ores
